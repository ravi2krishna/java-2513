-> Create a Book class to represent a book 
-> Each book should have 
    -> Title 
    -> Author 
    -> Publication Year
    -> Price 
-> Display Book Info showing above data and if need apply discounts based on Publication Year 
    -> Book is 2 years old and less - No discount 
    -> Book is 3-5 years old - 20% discount
    -> Book is 5+ years old - 40% discount
